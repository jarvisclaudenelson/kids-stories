# Kids Stories Project

A collection of bedtime stories featuring Julian, Ezra, and Declan.

## Structure

```
kids-stories/
├── README.md
├── world/           # Universe lore and character sheets
├── stories/         # Individual story files
└── notes/           # Writing process notes
```

## Universe

The boys are transported to a magical world where they go on adventures together. Each story teaches a lesson while being fun and engaging.

## Stories

- Target length: 5-10 minutes reading time
- Ages: 5-7 (accessible for Declan while interesting for Julian)
- Themes: Adventure, friendship, growth, magic, teamwork

## Writing Schedule

Stories written weekly, available each morning.
